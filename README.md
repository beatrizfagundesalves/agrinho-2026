# agrinho-2026
IntroduçãoO projeto "Agro Forte, Natureza Protegida" foi desenvolvido para o Concurso Agrinho 2026 como um site educativo focado no agronegócio sustentável. A plataforma utiliza um design moderno, com destaque para a imagem de um agricultor na lavoura de soja, ilustrando que a produção de alimentos e o crescimento econômico devem caminhar de forma unida com a preservação ambiental.

COM FUNCIONA 
O site funciona de forma simples e visual: abre com um topo verde-escuro e exibe no centro a imagem de um agricultor na lavoura de soja. Para garantir uma leitura confortável, o texto principal usa letras pretas sobre uma camada branca transparente que não se mistura com o desenho. No meio da página, o botão "Saiba mais" abre uma janela flutuante com explicações detalhadas sobre tecnologia e manejo do solo. Por fim, na parte inferior e fora da imagem de fundo, o usuário estuda sobre sustentabilidade através de cards brancos com perguntas e respostas diretas.
